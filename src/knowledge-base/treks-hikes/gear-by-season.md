# Trekking Gear Guide by Season

## The Layer System
Layering lets you regulate body temperature by adding or removing layers. Always use three layers:
1. **Base Layer**: Moisture-wicking (merino wool or synthetic). Keeps sweat off your skin.
2. **Mid Layer**: Insulation (fleece or down jacket). Traps body heat.
3. **Outer Layer**: Shell (waterproof, windproof jacket). Protects from rain and wind.

## Summer Treks (March - May)

| Item | Recommendation |
|------|---------------|
| Base layer | Lightweight synthetic T-shirts (Decathlon, Wildcraft) |
| Mid layer | Thin fleece (for higher altitudes) |
| Outer layer | Windbreaker |
| Bottoms | Quick-dry trek pants, shorts |
| Footwear | Ankle-high trekking shoes (Quechua MH500) |
| Sun protection | Wide-brim hat, UV sunglasses, SPF 50 sunscreen |
| Extras | Insect repellent, hydration pack |

## Monsoon Treks (June - September)

| Item | Recommendation |
|------|---------------|
| Base layer | Quick-dry synthetic |
| Outer layer | Waterproof rain jacket (sealed seams essential) |
| Bottoms | Quick-dry pants + rain pants |
| Footwear | Waterproof trekking boots with good grip (Vibram sole) |
| Backpack | Rain cover + dry bags for electronics |
| Extras | Gaiters, extra socks (2-3 pairs), waterproof phone pouch |

**Warning**: Leeches are common on monsoon treks. Wear gaiters and apply salt or DEET-based repellent around shoes.

## Post-Monsoon (October - November)

The best trekking season in India. Clear skies, moderate temperatures.

| Item | Recommendation |
|------|---------------|
| Base layer | Lightweight synthetic or merino |
| Mid layer | Fleece jacket |
| Outer layer | Light shell jacket |
| Footwear | Standard trekking shoes |
| Extras | Trekking poles for descents |

## Winter Treks (December - February)

| Item | Recommendation |
|------|---------------|
| Base layer | Merino wool thermal top and bottom |
| Mid layer | Down jacket (600+ fill power) or heavy fleece |
| Outer layer | Waterproof hardshell jacket and pants |
| Footwear | Insulated waterproof boots, crampons for snow |
| Hands/Head | Insulated gloves + liner gloves, balaclava, warm beanie |
| Sleeping | -10C to -20C rated sleeping bag (depending on altitude) |
| Extras | Hand warmers, thermos with hot water, snow goggles |
