# Training Plans for Running Events

## Couch to 5K (8 Weeks)
For complete beginners with no running background.

| Week | Plan |
|------|------|
| 1-2 | Alternate 1 min running / 2 min walking for 20 min, 3 days/week |
| 3-4 | Alternate 2 min running / 1 min walking for 25 min, 3 days/week |
| 5-6 | Run 5 min, walk 1 min, repeat for 30 min, 3 days/week |
| 7 | Run 20 min continuously, walk 5 min, run 10 min |
| 8 | Run 30 min (approx 5K) continuously |

**Key**: Run at a conversational pace. If you cannot talk, you are going too fast.

## 10K Preparation (6 Weeks)
Assumes you can already run 5K comfortably.

- **3 runs per week**: 1 easy run, 1 interval/tempo run, 1 long run
- **Week 1-2**: Easy 5K, intervals (6 x 400m with 90s rest), long run 6-7 km
- **Week 3-4**: Easy 5K, tempo run (3 km at race pace), long run 8-9 km
- **Week 5**: Easy 4K, intervals, long run 10 km
- **Week 6**: Taper -- easy 3K, easy 3K, race day

## Half Marathon (8 Weeks)
Assumes a base of 20-25 km per week and ability to run 10K.

- **4 runs per week**: 2 easy, 1 tempo/intervals, 1 long run
- Build long run by 2 km per week: 12, 14, 16, 18 km
- Include one 21 km practice run in week 6
- Taper in final 2 weeks -- reduce volume by 40%
- **Critical**: Do not increase weekly distance by more than 10%

## Trail Running Transition
Moving from road to trail requires different skills:

- **Start slow**: Trail pace is 20-30% slower than road pace. This is normal
- **Foot placement**: Watch the ground 2-3 metres ahead. Quick, short steps on technical terrain
- **Uphill**: Shorten stride, lean slightly forward, power-hike steep sections (no shame in walking)
- **Downhill**: Lean slightly forward (not back), quick feet, let gravity do the work
- **Gear**: Trail shoes with aggressive lugs (Salomon Speedcross, Asics FujiTrabuco), hydration vest for runs over 10 km
- **Practice**: Run on uneven surfaces -- parks with dirt paths, rocky areas, sand
- **Safety**: Carry a phone, tell someone your route, start with well-marked trails before exploring remote ones
